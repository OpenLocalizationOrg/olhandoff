#컨테이너에 대 한 PowerShell

##ContainerOSImage 설치

**NAME**      
ContainerOSImage 설치

**개요**  
Windows Server 또는 Hyper-v 컨테이너와 함께 사용에 대 한 컨테이너 OS 이미지 지정된 WIM은 설치합니다.


**구문**
``` PowerShell  
Install-ContainerOSImage [-WimPath] <String> [-Force] [< CommonParameters >]
```

**설명**  
Windows Server 및 Hyper-v 컨테이너 기능에 대 한 공유 중앙 이미지 저장소에 있는 WIM 파일에서 기본 이미지를 설치합니다.

**PARAMETERS**
``` PowerShell
    -WimPath <String>
        A path to the WIM file that will be installed.

        Required?                    true
        Position?                    1
        Default value
        Accept pipeline input?       false
        Accept wildcard characters?  false

    -Force [<SwitchParameter>]

        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false

    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (http://go.microsoft.com/fwlink/?LinkID=113216).
```

**입력**      
없음

**출력**  
없음
